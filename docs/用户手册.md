## 一、系统概述

### 1.1 系统简介

学生综合服务与教学协同平台是一个集教务管理、教师工作台、学生服务于一体的综合性教育管理平台。系统支持多角色访问，实现课程管理、成绩录入、科研申报、竞赛审批等核心功能。

### 1.2 主要功能模块

| 模块 | 说明 | 适用角色 |
|------|------|---------|
| **系统管理** | 用户管理、部门管理、权限配置、登录日志 | 管理员 |
| **课程管理** | 课程库维护、课程发布、选课管理 | 管理员、教务老师 |
| **成绩管理** | 成绩录入、审核、查询 | 教师、教务老师、学生 |
| **科研管理** | 科研申报、审核、档案管理 | 教师、教务老师 |
| **竞赛管理** | 竞赛申报、审批、获奖记录 | 学生、教师、教务老师 |
| **人事档案** | 教职工档案管理、学历信息管理 | 管理员 |
| **个人中心** | 个人信息查看与维护 | 所有用户 |

### 1.3 技术特点

- **统一身份认证：** 基于 JWT Token 的安全认证机制
- **RBAC 权限模型：** 细粒度的角色权限控制
- **数据隔离：** 不同角色只能访问授权范围内的数据
- **实时统计：** 首页展示关键业务数据统计卡片

## 二、快速入门

### 2.1 访问系统

**访问地址：** http://localhost:8080/index.html

## 三、角色与权限

### 3.1 角色体系

系统采用 RBAC（基于角色的访问控制）模型，主要角色包括：

| 角色 | userType | 数据范围 | 核心职责 |
|------|----------|---------|---------|
| **系统管理员** | '2' | 全局 | 系统配置、用户管理、权限分配 |
| **教务老师** | '1' + 特殊权限 | 全院系 | 课程发布、成绩审核、科研审核 |
| **普通教师** | '1' | 所授课程 | 教学管理、成绩录入、科研申报 |
| **学生** | '0' | 仅自己 | 选课、成绩查询、竞赛申报 |

### 3.2 权限标识说明

| 权限标识 | 说明 | 授予角色 |
|---------|------|---------|
| `academic:course:release` | 课程发布权限 | 教务老师 |
| `academic:score:input` | 成绩录入权限 | 教师 |
| `academic:score:audit` | 成绩审核权限 | 教务老师 |
| `research:view` | 科研管理权限 | 教师、教务老师 |
| `student:list` | 学生名单查看权限 | 教师、辅导员 |

### 3.3 菜单动态显示

系统根据用户权限动态显示菜单项：

- **普通教师：** 首页、我的课程、成绩管理、学生名单、科研申报
- **教务老师：** 首页、发布课程、科研审核、竞赛审批
- **学生：** 首页、个人信息、我的课程、我的成绩、选课中心、课表查询、信息维护、竞赛申报
- **管理员：** 首页、学生管理、课程管理、教师管理、人事档案、部门管理、权限查看、登录日志

## 四、管理员操作指南

### 4.1 首页 - 系统概览

**功能说明：** 展示系统整体运行数据统计

**显示内容：**
- 学生总数、教师总数、课程总数、部门总数
- 人事档案统计（已建档/未建档人数）
- 登录统计（今日登录次数、今日登录人数、总登录次数、总登录人数）

### 4.2 学生管理

#### 4.2.1 查看学生列表

1. 进入"学生管理"菜单
2. 在搜索框输入学号或姓名进行查询
3. 点击"查询"按钮筛选结果
4. 点击"重置"清空搜索条件

#### 4.2.2 新增学生

1. 点击右上角"新增学生"按钮
2. 填写学生信息表单：
   - **基本信息：** 学号（纯数字）、姓名、性别、出生日期、民族、身份证号、政治面貌
   - **联系方式：** 手机号、邮箱、生源地、家庭地址
   - **学籍信息：** 院系、专业、年级、班级、学历层次、入学年份
3. 点击"确定"保存

**必填字段：** 学号、姓名、手机号、邮箱、政治面貌

#### 4.2.3 编辑学生

1. 在学生列表中找到目标学生
2. 点击"编辑"按钮
3. 修改需要更新的信息
4. 点击"确定"保存

#### 4.2.4 删除学生

1. 在学生列表中找到目标学生
2. 点击"删除"按钮
3. 确认删除操作

> **警告：** 删除操作不可恢复，请谨慎操作。如果学生已有选课或成绩记录，系统将阻止删除。

### 4.3 课程管理

#### 4.3.1 查看课程列表

1. 进入"课程管理"菜单
2. 查看所有课程基础信息（课程代码、课程名称、学分、学时）

#### 4.3.2 新增课程

1. 点击"新增课程"按钮
2. 填写课程信息：
   - 课程代码（唯一标识）
   - 课程名称
   - 学分
   - 学时
3. 点击"确定"保存

#### 4.3.3 编辑课程

1. 在课程列表中找到目标课程
2. 点击"编辑"按钮
3. 修改课程信息
4. 点击"确定"保存

#### 4.3.4 删除课程

1. 在课程列表中找到目标课程
2. 点击"删除"按钮
3. 确认删除操作

> **注意：** 删除课程时，系统会自动清理该课程的排课记录和学生选课记录，确保数据一致性。

### 4.4 教师管理

#### 4.4.1 查看教师列表

1. 进入"教师管理"菜单
2. 查看所有教师基本信息

#### 4.4.2 新增教师

1. 点击"新增教师"按钮
2. 填写教师信息：
   - 工号（纯数字）
   - 姓名
   - 性别
   - 手机号
   - 邮箱
   - 所属部门
3. 点击"确定"保存

#### 4.4.3 编辑教师

1. 在教师列表中找到目标教师
2. 点击"编辑"按钮
3. 修改教师信息
4. 点击"确定"保存

#### 4.4.4 删除教师

1. 在教师列表中找到目标教师
2. 点击"删除"按钮
3. 确认删除操作

### 4.5 人事档案

#### 4.5.1 档案管理

1. 进入"人事档案"菜单
2. 查看教职工档案列表
3. 支持按姓名、工号搜索
4. 可查看档案详细信息：
   - 基本信息（姓名、性别、出生日期、身份证号等）
   - 工作信息（入职日期、职称、职务、所属部门等）
   - 学历信息（毕业院校、专业、学位、学习时间等）
   - 工作经历

#### 4.5.2 新建档案

1. 点击"新建档案"按钮
2. 选择教职工（从已有教师中选择）
3. 填写详细档案信息
4. 点击"确定"保存

#### 4.5.3 编辑档案

1. 在档案列表中找到目标档案
2. 点击"编辑"按钮
3. 修改档案信息
4. 点击"确定"保存

### 4.6 部门管理

#### 4.6.1 部门层级结构

系统支持三级部门结构：
- **学校级**（deptType = 0）
- **学院级**（deptType = 1）
- **系级**（deptType = 2）

#### 4.6.2 查看部门列表

1. 进入"部门管理"菜单
2. 查看所有部门信息（部门名称、部门类型、负责人、联系电话）

#### 4.6.3 新增部门

1. 点击"新增部门"按钮
2. 填写部门信息：
   - 部门名称
   - 部门类型（学校/学院/系）
   - 上级部门（可选）
   - 负责人
   - 联系电话
3. 点击"确定"保存

#### 4.6.4 编辑部门

1. 在部门列表中找到目标部门
2. 点击"编辑"按钮
3. 修改部门信息
4. 点击"确定"保存

#### 4.6.5 删除部门

1. 在部门列表中找到目标部门
2. 点击"删除"按钮
3. 确认删除操作

> **注意：** 如果部门下存在子部门或关联用户，系统将阻止删除。

### 4.7 权限查看

#### 4.7.1 按角色查看

1. 进入"权限查看"菜单
2. 选择"按角色查看"标签页
3. 选择要查看的角色
4. 查看该角色下的所有用户列表

#### 4.7.2 按用户查看

1. 进入"权限查看"菜单
2. 选择"按用户查看"标签页
3. 选择要查看的用户
4. 查看该用户的所有角色和权限

### 4.8 登录日志

#### 4.8.1 查看登录日志

1. 进入"登录日志"菜单
2. 查看所有用户的登录记录
3. 支持按用户名、登录时间范围筛选

#### 4.8.2 日志信息

每条登录日志包含：
- 用户名/账号
- 真实姓名
- 用户类型（学生/教师/管理员）
- 登录时间
- IP 地址
- 浏览器信息
- 登录状态（成功/失败）
- 失败原因（如果登录失败）

## 五、教师操作指南

### 5.1 首页 - 工作台概览

#### 5.1.1 普通教师首页

**教学统计：**
- 授课门数：本学期所授课程数量
- 学生总数：所有课程的学生总数
- 待录入成绩：需要录入的成绩数量
- 已录入成绩：已完成录入的成绩数量

**科研统计：**
- 科研通过：已通过审核的科研申请数量
- 科研待审核：等待审核的科研申请数量
- 科研被打回：被拒绝的科研申请数量

#### 5.1.2 教务老师首页

**课程管理：**
- 发布课程数：已发布的课程数量

**科研审核：**
- 科研待审核：等待审核的科研申请数量
- 科研已审核：已完成审核的科研申请数量

**竞赛审批：**
- 竞赛待审批：等待审批的竞赛申报数量
- 竞赛已审批：已完成审批的竞赛申报数量

### 5.2 我的课程 / 发布课程

#### 5.2.1 普通教师 - 查看我的课程

1. 进入"我的课程"菜单
2. 查看本学期所授课程列表
3. 每门课程显示：
   - 课程代码、课程名称
   - 学分、学时
   - 容量、已选人数
4. 点击"查看学生"可查看该课程的学生名单

#### 5.2.2 教务老师 - 发布课程

1. 进入"发布课程"菜单
2. 点击"发布课程"按钮
3. 从课程库中选择要发布的课程（可多选）
4. 为每门课程配置：
   - **授课教师：** 从教师列表中选择
   - **容量：** 设置最大选课人数（1-200）
   - **上课时间：** 设置上课时间段
   - **上课地点：** 设置教室位置
5. 点击"确定"批量发布

> **注意：** 同一课程在同一学期只能发布一次，重复发布将被系统拒绝。

#### 5.2.3 教务老师 - 编辑已发布课程

1. 在课程列表中找到目标课程
2. 点击"编辑"按钮
3. 修改课程配置信息
4. 点击"确定"保存

#### 5.2.4 教务老师 - 删除已发布课程

1. 在课程列表中找到目标课程
2. 点击"删除"按钮
3. 确认删除操作

> **警告：** 删除已发布课程将同时删除该课程的选课记录，请谨慎操作。

### 5.3 成绩管理（普通教师）

#### 5.3.1 选择课程

1. 进入"成绩管理"菜单
2. 从下拉列表选择要录入成绩的课程
3. 系统自动加载该课程的学生名单

#### 5.3.2 录入成绩

**单条录入：**
1. 在学生列表中逐行录入成绩
2. 分别输入：
   - **平时成绩**（占比 30%）
   - **期中成绩**（占比 20%）
   - **期末成绩**（占比 50%）
3. 系统自动计算总评成绩
4. 点击"保存草稿"暂存（可继续修改）
5. 点击"提交审核"提交给教务老师审核

**批量录入：**
1. 点击"批量录入"按钮
2. 在弹出的对话框中，以表格形式批量输入成绩
3. 点击"批量保存"一次性保存所有成绩

#### 5.3.3 成绩状态

- **草稿：** 可自由修改，学生不可见
- **待审核：** 已提交，等待教务老师审核，教师不可修改
- **已发布：** 审核通过，学生可见，不可修改
- **已打回：** 审核不通过，需修改后重新提交

### 5.4 学生名单（普通教师）

#### 5.4.1 查看学生名单

1. 进入"学生名单"菜单
2. 从下拉列表选择课程
3. 查看该课程的学生列表
4. 每个学生显示：
   - 学号、姓名
   - 班级、专业
   - 手机号、邮箱
   - 学籍状态

#### 5.4.2 导出名单

1. 选择课程后，点击右上角"导出名单"按钮
2. 系统将生成 CSV 格式文件
3. 保存到本地

### 5.5 科研申报（普通教师）

#### 5.5.1 提交科研申请

1. 进入"科研申报"菜单
2. 点击"新增申报"按钮
3. 选择科研类型：
   - **SCI 论文**
   - **卓越期刊论文**
   - **CCF 会议论文**
   - **科研奖励**
   - **科研项目**

#### 5.5.2 填写申报信息

**SCI 论文：**
- 论文题目、发表期刊
- 作者信息、通讯作者
- SCI 分区、CCF 分区
- 是否推荐期刊、卓越领军期刊
- 青大是否第一单位
- 是否高被引、是否 ESI 论文
- 奖励级别、论文归属、积分

**卓越期刊论文：**
- 论文题目、发表期刊
- 作者信息、通讯作者
- 期刊类型（领军/卓越）
- 青大是否第一单位
- 论文归属、积分

**CCF 会议论文：**
- 论文题目、会议名称
- 一作、通信作者
- 会议类型（CCF A/B/C）
- 青大是否第一单位
- 论文归属、积分

**科研奖励：**
- 奖励名称、成果名称
- 获奖级别、获奖等级
- 获奖日期、发证机关、证书号
- 所有完成人、单位排名、主要成果

**科研项目：**
- 项目编号、项目名称
- 项目来源、负责人
- 签署年度、合同签订/截止日期
- 合同金额、到账金额

4. 填写备注（可选）
5. 点击"提交"发送审核

#### 5.5.3 查看申请历史

1. 在科研申报页面查看我的申请列表
2. 每条申请显示：
   - 申请标题、科研类型
   - 提交时间
   - 审核状态（待审核/已通过/已拒绝）
   - 审核意见（如果已审核）
3. 点击"查看详情"查看完整信息

#### 5.5.4 撤回申请

- 仅在"待审核"状态下可撤回
- 点击"撤回"按钮即可

### 5.6 科研审核（教务老师）

#### 5.6.1 查看待审核列表

1. 进入"科研审核"菜单
2. 查看所有待审核的科研申请
3. 可按科研类型、申请人筛选

#### 5.6.2 审核申请

1. 在申请列表中找到目标申请
2. 点击"查看详情"查看完整信息
3. 系统根据科研类型智能展示结构化详情（非原始 JSON）
4. 点击"通过"或"拒绝"按钮
5. 如果拒绝，需填写拒绝原因
6. 确认审核操作

#### 5.6.3 查看审核历史

1. 切换至"已审核"标签页
2. 查看历史审核记录
3. 可查看审核时间和审核意见

### 5.7 竞赛审批（教务老师）

#### 5.7.1 查看待审批列表

1. 进入"竞赛审批"菜单
2. 查看所有待审批的竞赛申报
3. 可按申报时间、学生姓名筛选

#### 5.7.2 审批竞赛

1. 在申报列表中找到目标申报
2. 点击"查看详情"查看申报信息
3. 点击"通过"或"拒绝"按钮
4. 如果拒绝，需填写拒绝原因
5. 确认审批操作

## 六、学生操作指南

### 6.1 首页 - 学生工作台

**学习统计：**
- 课程总数：已选课程数量
- 本周待上课数：本周需要上的课程数量

### 6.2 个人信息

#### 6.2.1 查看个人信息

**学籍信息：**

- 学籍状态（在读/休学/退学/毕业）

**紧急联系人：**
- 紧急联系人姓名

**联系方式与地址：**
- 手机号、邮箱
- 家庭地址

### 6.3 我的课程

#### 6.3.1 查看已选课程

1. 进入"我的课程"菜单
2. 查看本学期已选课程列表
3. 每门课程显示：
   - 课程代码、课程名称
   - 学分、学时
   - 授课教师
   - 上课时间、上课地点
   - 容量、已选人数

### 6.4 我的成绩

#### 6.4.1 查看成绩

1. 进入"我的成绩"菜单

3. 每门课程显示：
   - 课程名称、授课教师
   - 平时成绩、期中成绩、期末成绩
   - 总评成绩
   - 成绩状态（已发布）

### 6.5 选课中心

#### 6.5.1 浏览可选课程

1. 进入"选课中心"菜单
2. 查看当前学期已发布的可选课程

   - 课程代码、课程名称
   - 学分、学时
   - 授课教师
   - 容量、已选人数、剩余名额
   - 上课时间、上课地点

#### 6.5.2 选课

1. 在可选课程列表中找到目标课程
2. 点击"选课"按钮
3. 系统检查：

   - 上课时间是否冲突

4. 如果检查通过，选课成功
5. 如果检查失败，系统提示具体原因

#### 6.5.3 退课

1. 在已选课程列表中找到目标课程
2. 点击"退课"按钮
3. 确认退课操作

> **注意：** 退课后该课程名额将释放给其他学生。超过选课截止时间后无法退课。

#### 6.6.1 查看课表

1. 进入"课表查询"菜单
2. 选择周次（1-18 周）

   - 课程名称
   - 上课时间
   - 上课地点
   - 授课教师

   - 手机号
   - 邮箱
   - 家庭地址
   - 紧急联系人
   - 紧急联系电话
4. 点击"保存"提交修改

> **注意：** 学号、姓名、身份证号等关键信息不可自行修改，如需变更请联系管理员。

### 6.8 竞赛申报

#### 6.8.1 提交竞赛申报

1. 进入"竞赛申报"菜单
2. 点击"新增申报"按钮
3. 填写申报信息：
   - 竞赛名称

#### 6.8.3 撤回申报

- 点击"撤回"按钮即可

## 七、常见问题

### 7.1 登录问题

**Q1: 忘记密码怎么办？**  
A: 请联系系统管理员重置密码。管理员可在"学生管理"或"教师管理"中直接修改用户密码。

**Q2: 提示"账号或密码错误"？**  
A: 
- 检查账号是否正确（学号/工号为纯数字）
- 检查密码大小写
- 确认账号是否已被停用
- 如仍无法登录，联系管理员

**Q3: 登录后提示"无权访问"？**  
A: 您的账号可能未被分配相应角色或权限，请联系管理员分配权限。

### 7.2 课程相关问题

**Q4: 为什么我看不到某些课程？**  
A: 
- 学生：只能看到当前学期已发布的可选课程
- 教师：只能看到自己授课的课程
- 课程可能尚未发布或已关闭

**Q5: 选课时提示"课程容量已满"？**  
A: 该课程已达到最大选课人数，无法继续选课。建议选择其他课程或等待有人退课。

**Q6: 选课时提示"上课时间冲突"？**  
A: 您选择的课程与已选课程的上课时间有重叠，请先退掉冲突的课程再选课。

**Q7: 为什么我无法退课？**  
A: 
- 可能已超过选课截止时间
- 该课程已开始授课
- 请联系教务老师处理

### 7.3 成绩相关问题

**Q8: 为什么我看不到某门课程的成绩？**  
A: 
- 成绩可能尚未录入
- 成绩可能处于"待审核"状态
- 成绩可能被"打回"需要教师重新录入
- 请耐心等待教师录入和教务审核

**Q9: 成绩录入后还能修改吗？**  
A: 
- "草稿"状态：可以自由修改
- "待审核"状态：教师无法修改，需联系教务老师退回
- "已发布"状态：无法修改，如需更正请联系教务老师

**Q10: 总评成绩是如何计算的？**  
A: 总评成绩 = 平时成绩 × 30% + 期中成绩 × 20% + 期末成绩 × 50%

### 7.4 科研申报问题

**Q11: 科研申报提交后还能修改吗？**  
A: 
- "待审核"状态：可以撤回后重新提交
- "已通过"或"已拒绝"状态：无法修改，如需补充材料请联系教务老师

**Q12: 科研审核需要多长时间？**  
A: 通常在 3-5 个工作日内完成审核，请耐心等待。如长时间未审核，可联系教务老师催促。

**Q13: 科研申请被拒绝怎么办？**  
A: 
- 查看拒绝原因
- 根据意见修改申报材料
- 重新提交申请

### 7.5 竞赛申报问题

**Q14: 竞赛申报后可以更换团队成员吗？**  
A: 
- "待审批"状态：可以撤回后重新提交
- "已通过"状态：无法自行修改，请联系教务老师

**Q15: 竞赛审批需要多长时间？**  
A: 通常在 3-5 个工作日内完成审批，请耐心等待。

### 7.6 其他问题

**Q16: 页面显示异常或功能无法使用？**  
A: 
- 尝试刷新页面（Ctrl + F5 强制刷新）
- 清除浏览器缓存
- 更换浏览器重试
- 如仍无法解决，联系技术支持

**Q17: 如何导出学生名单或成绩？**  
A: 
- 学生名单：教师在"学生名单"页面点击"导出名单"
- 成绩：暂未开放导出功能，请联系管理员

**Q18: 个人信息中的关键信息（如姓名、身份证号）如何修改？**  
A: 关键信息不可自行修改，请联系管理员或教务老师协助修改。
